Grid Structure

The grid is essentially one column for the main content, but on a desktop screen, to the left of the main content is a sidebar with major links, and to the right of the main content is a sidebar with social media links.

When screen size is decreased, the grid is a true one-column grid where the left sidebar serves as a top header and the right sidebar becomes the footer.

The homepage (projects page) takes the full width and height of the available screensize on desktop, having a non-scrolling effect so all content is visible, and project cards split the page horizontally in half.
